# Categories

The 13 repositories in Cyber Port are organized into five security domains.

---

## Threat Detection & Response

| Repository | Primary Focus |
|------------|--------------|
| honeypot-foundry | Deception, honeypots, attack telemetry collection |
| dfir-attack-lab | Digital forensics, incident triage, timeline analysis |
| ir-playbooks-automation | Incident response, playbooks, containment automation |

| Repository | Primary Focus |
