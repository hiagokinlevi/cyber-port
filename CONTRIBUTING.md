# Contributing to Cyber Port

## Before You Start

- Read [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)
- All contributions must be in English
- All tools and content must be strictly defensive
- Identify the correct existing repository for your contribution

## Process

1. Fork the target repository (one of the 13 base repos)
2. Create a feature branch: `git checkout -b feat/your-feature`
3. Implement with tests (coverage ≥ 70%)
4. Open a pull request with a clear description

## Governance Rule

Do not create new repositories. All additions go inside the 13 existing base repos.
