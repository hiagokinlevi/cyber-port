# Security Policy

## Scope

This repository is the root index for the Cyber Port portfolio. If you find a security vulnerability in any of the 13 base repositories, please report it via that repository's GitHub Security Advisory.

## Responsible Use

All tools in this portfolio are for authorized, defensive use only. Misuse is a violation of the license and may be illegal.

## Reporting

To report a vulnerability in any Cyber Port repository:
1. Go to the relevant repository on GitHub
2. Navigate to Security → Report a vulnerability
3. Provide a clear description and reproduction steps
4. Do not disclose publicly until a fix is available
