# Cyber Port

A public cybersecurity portfolio by [Hiago Kin Levi](https://github.com/hiagokinlevi) — a curated collection of 13 open-source repositories covering defensive security, cloud security, DevSecOps, AI security, and more.

All repositories are free to use, adapt, and share under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/) with attribution.

---

## Vision

Provide professionals, students, and security teams with production-quality, modular, and well-documented tools that cover the full spectrum of modern defensive security — from honeypots to incident response, from WAF rules to AI guardrails.

Everything published here is strictly defensive, authorized, and ethical.

## Governance

**13 permanent base repositories. No new repos will be created.**

New content always goes *inside* one of the 13 existing repositories as a module, script, template, playbook, or training material. This keeps the portfolio coherent, navigable, and professionally maintained.

See [CATEGORIES.md](CATEGORIES.md) for the full taxonomy and [STANDARDS.md](STANDARDS.md) for contribution and code standards.

Live portfolio completion tracking is published in [PORTFOLIO_STATUS.md](PORTFOLIO_STATUS.md).
The execution policy for finishing the portfolio lives in [COMPLETION_STRATEGY.md](COMPLETION_STRATEGY.md).

## How to Contribute

1. Read [CONTRIBUTING.md](CONTRIBUTING.md) and [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)
2. Choose the repository that matches your contribution domain
3. Open an issue first for significant additions
4. Submit a PR with tests and documentation

All contributions must be in **English**, strictly **defensive**, and meet the professional standards defined in [STANDARDS.md](STANDARDS.md).

## Ethical Disclaimer

All tools and content in this portfolio are designed exclusively for authorized, defensive security work. Never use these tools against systems you do not own or have explicit written permission to test or monitor. Unauthorized access to computer systems is illegal. Always verify compliance with applicable laws and organizational policies before deployment.
